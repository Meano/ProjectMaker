# Project Maker

**A Shell tool to build custom projects by clearing predefined comment tags.**

## Supported File Types: 
 - ***.sln**
 - ***.cs**
 - ***.xml**
 - ***.xaml**
 - ***.c**
 - ***.h**

## Dependencies:
 - bash
 - find
 - sed

## Usage: 
```bash
pmake <ProjectName> <CustomTags> <CustomFiles> [BuildTag]
```
